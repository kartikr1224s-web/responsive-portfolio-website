## Overview

Have you ever wanted to explore a vast collection of movies and TV shows all in one place? Look no further than MovieLand! This web application allows users to browse, search, and discover a wide range of films and series with ease. You can see the name of the movie, poster image, year of release and search for your favorite titles. Whether you're a cinephile or just looking for something new to watch, MovieLand has got you covered!

## Features

- Browse a comprehensive collection of movies and TV shows.
- Search for specific titles using the search bar.
- View year of release and poster images for each movie.

> This is a project built for learning purposes using React.

All movie data is fetched from the [OMDb API](http://www.omdbapi.com/).
